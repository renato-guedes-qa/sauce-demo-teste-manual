# SauceDemo

Projeto de Quality Assurance aplicado ao **SauceDemo MyShopify**, simulando o processo de documentação e execução de testes em um ambiente de e-commerce.

Este repositório faz parte do meu portfólio em Quality Assurance e reúne artefatos produzidos durante o planejamento, execução e evolução do projeto.

---

## 📌 Objetivo

Documentar um projeto de QA de ponta a ponta, contemplando atividades comuns ao dia a dia de um Analista/QA Engineer, desde o entendimento das regras de negócio até a automação dos testes.

---

## 📂 Estrutura do projeto

```
📁 docs/
   └── Plano de Testes.pdf

📁 evidencias/
   └── Capturas de tela da execução dos testes

📁 bugs/
   └── Relatórios de defeitos encontrados

📁 automacao/
   └── Scripts de automação com Playwright

README.md
```
